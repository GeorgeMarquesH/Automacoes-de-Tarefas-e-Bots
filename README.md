# 🤖 Automações de Tarefas e Bots

## 📌 Descrição
Este projeto demonstra como automatizar tarefas repetitivas usando **Python** e a biblioteca **PyAutoGUI**.  
O objetivo é simular o processo de login em um sistema da empresa e cadastrar produtos de forma automática.

---

## 🚀 Passo a passo da automação
1. Entrar no sistema da empresa via navegador.
2. Fazer login com usuário e senha.
3. Abrir a base de dados (`produtos1.csv`).
4. Cadastrar cada produto automaticamente no sistema.
5. Repetir o processo até finalizar a lista.

---

## 🛠️ Tecnologias utilizadas
- Python  
- PyAutoGUI  
- Pandas  
- Time  

---

## 📊 Resultado
Login automatizado e cadastro de todos os produtos da planilha sem intervenção manual.
